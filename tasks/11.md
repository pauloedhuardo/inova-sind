## Tarefa

Refatorar a página Home para exibir um formulário para atualizar o telefone do usuário

## Requisitos Técnicos

- Deverá recuperar o usuário autenticado
- Utilize a função `updateUser` em @app\_lib\api\fetch-generated\index.ts para atualizar o telefone do usuário autenticado.
- Crie um formulário utilizando Card dentro de um Dialog (components\ui\dialog.tsx) com o campo "Telefone para contato".

## Regras de Negócio

- O dialog com o formulário de atualização do telefone do usuário autenticado deverá ser aberto caso `phoneVerified=false`.
- A página Home só será visualizada após a atualização do telefone do usuário autenticado.
- Atribua o valor "true" para o atributo "phoneVerified".
