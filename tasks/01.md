## Tarefa

Crie a tela de login e integre ela com a API.

## Requisitos Técnicos

- Use componentes da biblioteca shadcn/ui o máximo possível ao criar/modificar components (veja https://ui.shadcn.com/ para a lista de componentes disponíveis).

- Use a imagem @logo.png centralizada como o logotipo do app.
- Crie componentes responsivos e utilize a estratégia de design mobile firts.
- Crie um botão personalizado "Faça o login com o Google".
- Ao clicar no botão "Faça o login com o Google", chame o auth-client em @app\_lib\auth-client.ts para fazer login.ex
