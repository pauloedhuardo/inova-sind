## Tarefa

Crie um formulário de cadastro de Avaliações de uma organização

## Requisitos técnicos

- Crie o formulário em @app\assessment\page.tsx.
- Essa tela deve ser um server component
- Deverá receber um `organizationId` de uma organização como parâmetro de URL.
- Deverá ter um campo de busca code complete para buscar uma organização pelo nome.
- Utilize a função `getOrganizations`, em @app\_lib\api\fetch-generated\index.ts, para buscar os dados da organização.
- - Terá labels para visualização dos dados da organização que só serão renderizados se os dados existirem. Exemplo em @app\organization\link\page.tsx.
- O campo score será deverá ter o label "avaliação" e será um componente do tipo slider (@components\ui\slider.tsx) com valores de 1 a 10.
- O botão de enviar o formulário deve ser com o background na cor primária.
- Utilize o componente Card em @components\ui\card.tsx para agrupar os campos do formulário.
- Crie um cabeçalho na página do formulário utilizando o componente `header` em @components\header.tsx.
- Use o zod para validação dos campos do formulário.
- Utlize os estilos css em @app\globals.css.

## Regras de negócio

- Se o parâmetro `organizationId` não existir, renderizar o campo para buscar uma organização.
- O campo `user` deverá ser vinculado ao usuário autenticado.
- Ao finalizar a avaliação, deverá ser redirecionado para a página Home.
